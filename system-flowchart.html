五、核心流程技术细节

阶段技术说明
认证JWT7天过期，拦截器校验，排除游客路径
选座锁座Redis Lua原子性检查座位状态 + 锁定，15分钟 TTL
订单创建JPA关联 showtime + seats，生成唯一订单号
异步处理Redis Stream订单状态流转，超时自动取消
支付扣款乐观锁version 字段防并发，虚拟钱包 BigDecimal 计算
状态同步轮询前端轮询订单状态接口，实时更新 UI
想看功能Redis Set双向索引：movie:wish:{id} ↔ user:wish:{id}
票房排行Redis Cache定时同步，缓存今日/累计票房
图片上传Multipart本地存储 uploads/，前端 resolveImageUrl 拼接
密码加密BCrypt修改密码需校验旧密码
